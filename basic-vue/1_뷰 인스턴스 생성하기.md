## 🐫 뷰 인스턴스(객체) 만들기

* Vue객체는 ``<script>``에서 생성하여 사용할 수 있다.

  ```javascript
    <body>
      <div id="app">
        이름 : {{ name }}
      </div>
    
      <script src="https://cdn.jsdelivr.net/npm/vue"></script>
      <script>
        new Vue({
          el: "#app",
          data: {
            name: "코지 코더"
          }
        });
      </script>
    </body>
  ```
